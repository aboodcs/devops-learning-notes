# to bring the color of web application?
first thing we get the pods 
kubectl get pods 
then check the svc 
kubectl get svc
ex: webapp-service   NodePort   8080:30080/TCP we take the 30080
then make the curl http://controlplane:30080
Search only for the background color
curl -s http://controlplane:30080 | grep -i background

Update a deployment:
kubectl set image deploy frontend simple-webapp=kodekloud/webapp-color:v2

Check rollout status:
kubectl rollout status deployment/my-app

If something goes wrong, roll back:
kubectl rollout undo deployment/my-app
